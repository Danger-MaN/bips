# Bitcoin Dormant Recovery Proposal (BDRP)

This repository contains the draft of a proposed Bitcoin Improvement Proposal (BIP) titled "Bitcoin Dormant Recovery Proposal". It introduces a voluntary system for long-term Bitcoin holders to prove wallet activity using cryptographic signatures.

## Summary

- **Name**: Bitcoin Dormant Recovery Proposal (BDRP)

- **Author**: Mohamed Magdi Ahmed [Idris Magdi] (X.CoM/Page_Magdi)
- **Status**: Draft
- **Purpose**: Prevent total deflation by helping distinguish lost wallets (after 100+ years) using signed LTAP messages.
